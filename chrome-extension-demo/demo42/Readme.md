# 可供任何扩展程序页面使用的实用程序 展示 (chrome.extension)

## manifest.json 权限
```json
{
    "action": {
        "default_icon": "images/icon.png",
        "default_title": "展示 chrome.extension API"
    },
    "background": {
        "service_worker": "js/background.js"
    }
}
```
